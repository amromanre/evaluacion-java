Se va a agregar EclipseLink como proveedor JPA, manteniendo UUID como tipo de id y usando H2 como base de datos.

Pasos:
1. Se agregará la dependencia de EclipseLink en el pom.xml.
2. Se configurará application.properties para usar EclipseLink y la plataforma H2.

No se eliminará el uso de UUID en las entidades.
